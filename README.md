To run the Project:
1. Using Visual Studio 2022 : 
	a. Open .sln File
	b. Install Docker Desktop (if not already installed)
	c. Run using cntrl + F5 or F5
2. Deploy via docker
	a. Pulll git
	b. Navigate to root
	c. run docker-compose build
